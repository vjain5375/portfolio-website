## About

A modern, immersive portfolio website featuring Stranger Things-inspired dark aesthetics with atmospheric effects, 3D elements, and cinematic animations.

## Technologies Used

- **React** - UI Framework
- **TypeScript** - Type Safety
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI Components
- **Three.js** - 3D Graphics

## Getting Started

```sh
# Clone the repository
git clone https://github.com/vjain5375/portfolio-website.git

# Navigate to project directory
cd portfolio-website

# Install dependencies
npm install

# Start development server
npm run dev
```
